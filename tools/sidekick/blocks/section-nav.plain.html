name
Section Nav
description

A left-rail sub-navigation for a section of the site. The current page is marked active (highlighted with a teal left bar); items flagged [expand] show a chevron.

Implementation

One row per nav item, each a single cell containing a link. Prefix the cell with [active] to mark the current page and/or [expand] to show a chevron.
